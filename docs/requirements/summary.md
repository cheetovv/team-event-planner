Roles




Administrador

Puede:

Crear equipos.
Crear usuarios.
Asignar usuarios a equipos.
Nombrar coordinadores.
Crear eventos para cualquier equipo.
Invitar usuarios de cualquier equipo.
Ver todos los eventos.




Coordinador

Puede:

Crear eventos para su propio equipo.
Modificar eventos creados por él.
Invitar usuarios de cualquier equipo.
Ver eventos de su equipo.

No puede:

Crear equipos.
Administrar usuarios.
Crear eventos para otros equipos.




Empleado

Puede:

Ver eventos donde fue invitado.
Aceptar invitaciones.
Rechazar invitaciones.
Consultar su calendario.

No puede:

Crear eventos.
Invitar personas.
