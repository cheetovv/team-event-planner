# team-event-planner
Team event scheduling and calendar management system built with Java, Spring Boot and PostgreSQL.

Sistema para la gestión de eventos organizacionales y coordinación de equipos.

## Objetivos

- Gestionar equipos
- Crear eventos
- Administrar invitaciones
- Reservar espacios en el calendario
- Controlar permisos mediante roles

## Tecnologías

### Backend
- Java
- Spring Boot
- PostgreSQL

### Frontend
- HTML
- CSS
- JavaScript
